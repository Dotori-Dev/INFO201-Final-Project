# Military Spending vs GDP Dashboard

Live Dashboard: [View the Dashboard](https://dotori-dev.github.io/Military_Spending_vs_GDP_Dashboard/Military_Spending_vs_GDP_Dashboard.html)

This project explores the relationship between military spending and GDP per capita across countries. The goal is to see how economic strength connects with defense spending and how those patterns change across different nations and time periods.

The dashboard allows users to explore the data interactively and better understand global spending trends.

## Project Overview

Military spending often reflects a country's security priorities and geopolitical situation. GDP per capita reflects the average economic output per person and can indicate the economic strength of a country.

By combining these two measures, this dashboard helps show how countries balance economic growth with defense spending. Users can explore patterns, compare countries, and observe how spending changes across time.

## Interactive Dashboard

The dashboard includes interactive visualizations that allow users to explore the dataset. Users can hover over charts to see detailed values and compare different countries.

Interactive features were implemented using the plotly library.

## Tools and Technologies

- R  
- Quarto  
- ggplot2  
- plotly  

## Data Sources

The project uses publicly available international datasets on:

- Military spending as a percentage of GDP  
- GDP per capita by country and year  

These datasets were cleaned, joined, and processed before being used in the dashboard.
